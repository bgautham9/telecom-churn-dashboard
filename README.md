# 📊 Telecom Analytics Dashboard – Power BI

## 🚀 Project Overview

This project analyzes telecom customer behavior using Power BI.  
The dashboard provides insights into customer usage patterns, revenue generation, and churn behavior.

The goal was to transform a messy dataset into a professional **star schema data model** and build an interactive analytics dashboard.

---

## 📂 Dataset

Source: Simulated telecom customer dataset

Contains:
- Customer demographics
- Telecom plans
- Monthly data usage
- Call minutes
- Payment methods
- Contract types
- Churn status

---

## 🧠 Key Analysis

The dashboard answers questions such as:

- Which telecom plans generate the most revenue?
- Which states have the highest usage?
- Which customers consume the most data?
- What factors contribute to customer churn?

---
